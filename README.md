# Banking System

A simple banking system implemented in Java with support for multiple accounts and transactions.

## Features

- ✅ Create bank accounts
- ✅ Deposit money
- ✅ Withdraw money
- ✅ View account balance
- ✅ View transaction history
- ✅ Transfer money between accounts
- ✅ List all accounts

## Project Structure
src/SistemaBancario/
├── ContaBancaria.java       - Represents a single bank account
├── SistemaBancario.java     - Manages all bank accounts
└── SistemaBancarioMain.java - User interface and menu

## How to Run

1. Open the project in IntelliJ IDEA
2. Run the `SistemaBancarioMain` class
3. Follow the interactive menu

## Technologies Used

- Java 11+
- Object-Oriented Programming (OOP)
- ArrayList for data management
